Caratteristiche

  • Facilità d'uso (le impostazioni predefinite funzionano nella maggior parte dei casi)
  • Funziona con qualsiasi strumento
  • Precisione fino a un cent (0.01 semitoni)
  • Display in tempo reale
  • Ottave e note selezionabili
  • Trasposizione
  • Temperamento selezionabile
  • Internazionalizzazione
  • Cross-platform (può essere compilato praticamente in qualsiasi sistema operativo)
  • Documentazione di base inclusa
