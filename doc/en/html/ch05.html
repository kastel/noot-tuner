No note is detected

I.e. the Note, Frequency, and Offset fields do not show anything.

  • The volume may be too low or the threshold too high. Look at the volume meter in the lower-right corner and check that the gauge goes past the thin line when a note is produced. Check the input volume of your sound card.

  • You may have selected the wrong input device. If the volume meter does not move at all, select another input device in the Devices menu.

  • The expected precision could be too high for the window size selected. Select Auto as window size or raise the Expected precision value. Select Continuous power spectrum as refinement algorithm (from the Other options menu).

The wrong note is detected

  • Facilitate the work of the program by telling what note or at least what octave you are tuning. (Set the Note and Octave options.)

  • You may need extra precision: try raising the Window size.

The indicator glitches

  • Try setting the Octave and Note options. If the indicator does not stabilize, try raising the Window size and/or lowering the Expected precision option and/or raising the sampling rate. If you are tuning an extremely high or low note (as in the piano) follow the instructions in the section called “Tuning extremely high or low notes”.

The program crashes or freezes

This should not happen. Please file a bug.
