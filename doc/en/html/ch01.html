Never-out-of-tune Instrument Tuner simulates a full-featured instrument tuner. It may be used to tune musical instruments (guitars, pianos) with a cent precision.

Features

  • Easy to use (default settings will work in most cases)

  • It works with almost any instrument

  • Half-cent precision (0.005 semitones) or even better

  • Octave and/or note selection

  • Clock correction

  • Transposition

  • Clipping detection

  • Selectable temperament

  • Internationalisation (English and Italian for now, other languages will follow)

  • Cross-platform (it can be compiled on virtually any OS; Windows, Ubuntu and generic Linux packages available now, other packages available soon)

  • Basic documentation included

  • Two different frequency refinement algorithms

Future plans

  • Customisable temperament

  • More ports (OSX, ...) and languages

Minimum requirements

  • An audio card with a configured microphone or line input

  • Variable disk space (10 MB for the Windows version, 1 MB for Ubuntu packages, about 40 MB for universal packages)

Changelog

Version 0.3.0:

  • New frequency refinement algorithm (Continuous power spectrum), more precise and efficient (now default). The old algorithm is available in the Other options menu

  • Sample rate option is saved separately for the two refinement algorithms

  • New pitch detection algorithm: pitch detection is more reliable when Octave and Note are set to Auto

  • Higher reactivity with low sample rates

  • Added option Window type (rectangular or Hanning)

Version 0.2.1:

  • Support for offline analysis (sound file)

  • Added a text interface (for testing purposes)

Version 0.2.0:

  • Volume meter, clipping detection

  • Clock correction

  • No more hangs due to excessive use of the CPU

Version 0.1.3:

  • Improved documentation (new format)

Version 0.1.2.3:

  • Precision is now given in millis, not in cents

  • Minor improvements to the display

  • PPA for Ubuntu users

Version 0.1.0:

  • First published version
